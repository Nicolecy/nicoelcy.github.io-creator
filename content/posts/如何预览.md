# 创建个人GitHub仓库
新建一个GitHub repo,命名为 你的GitHub用户名.github.io 除了命名以外的选项都不用动

# 创建本地仓库
## 步骤
进入文件夹中的public
创建本地仓库，上传，提交
    
    git init
    git add .
    git commit -m hello
